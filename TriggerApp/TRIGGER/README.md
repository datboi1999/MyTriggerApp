# TRIGGER Growth Engine

A high-performance browser-based growth engine for influencers and musicians.

## Monorepo Structure

- `apps/frontend` — Next.js (React + TypeScript) UI
- `apps/backend` — Node.js (NestJS/Fastify) microservices
- `apps/ai-agents` — Python AI microservices (Humanizer, Optimizer, Stabilizer, Economist)
- `packages` — Shared libraries and SDK
- `infrastructure` — Deployment, IaC, and DevOps
- `docs` — Documentation

## Core Features
- Automated human-like engagement simulation
- Anti-Fluctuation Logic for stable growth
- Real-time analytics and dashboard
- Multi-platform campaign orchestration
- Secure, scalable, and cost-optimized

## Performance Targets
- Trigger queue latency: <150ms
- API response: <300ms
- Dashboard refresh: ≤3s

## Tech Stack
- Next.js, TailwindCSS, Zustand, Recharts, Framer Motion
- Node.js, NestJS/Fastify, PostgreSQL, Redis, Kafka/RabbitMQ
- Python, TensorFlow/PyTorch
- Vercel, AWS, Cloudflare, GitHub
